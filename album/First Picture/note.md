# First Picture

这是一张用于测试相册系统的图片。

我希望相册不只是把图片铺在页面上，而是让每一张图片都带一点情绪、观察或者随手记下来的想法。这样它才更像个人网站，而不是单纯的图库。

## 这次测试的重点

- 首页相册区是否能正确读取图片与随笔
- 点击图片后是否会在当前窗口弹出次级内部窗口
- 弹窗里的 Markdown 是否能渲染成清晰的阅读内容

## 一点随笔

页面的背景、信息密度和浏览节奏，其实都会影响一张图被观看的方式。  
如果背景本身已经很强，那前景信息就更需要克制、有边界，而且要有一点呼吸感。
